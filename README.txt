# Manzano Analytics

Welcome to the Manzano Analytics repository. This repository contains the source code for the Manzano Analytics website.

## Introduction

Manzano Analytics is a platform dedicated to data analysis and investment management. The website showcases various projects, insights, and expertise in the field of data analytics.

## Features

- **Project Portfolio**: Explore a diverse range of data analysis projects covering topics such as women's empowerment, marketing strategy, and more.
- **Skills and Expertise**: Discover the skills and tools utilized in data analysis projects, including Python, SQL, Tableau, and advanced visualization techniques.
- **Contact Information**: Connect with Aaron Manzano, the founder of Manzano Analytics, via email, LinkedIn, or GitHub.

## Installation

To run the Manzano Analytics website locally, follow these steps:

1. Clone this repository to your local machine.
2. Open the `index.html` file in your web browser.

## Project Structure

- `index.html`: Main HTML file containing the structure of the website.
- `assets/`: Directory containing CSS and JavaScript files for styling and functionality.
- `images/`: Directory containing images used in the website.
